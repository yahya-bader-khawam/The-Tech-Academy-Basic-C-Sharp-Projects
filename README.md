# C# & .NET Development Projects

This repository contains a collection of console applications and logic modules developed while mastering the fundamentals of C# and the .NET Framework.

## Projects & Drills

### Object-Oriented Programming (OOP) Suite
A series of projects demonstrating core OOP principles in C#:
* **Polymorphism & Interfaces:** Implemented an `IQuittable` interface to showcase polymorphic behavior by treating `Employee` objects as interface types.
* **Operator Overloading:** Enhanced the `Employee` class by overloading the `==` and `!=` operators to compare objects based on unique IDs.
* **Methods & Logic:** Created specialized classes to perform complex math operations using positional and named parameters.

### Package Express
A shipping quote console application that utilizes branching logic, user input validation, and decimal-precision calculations to determine shipping costs for customers.

## Technical Skills Gained
* **Languages:** C#
* **Frameworks:** .NET Framework / .NET Core
* **Concepts:** Inheritance, Encapsulation, Polymorphism, Abstraction, Interface Implementation, and Operator Overloading.
* **Tools:** Visual Studio, Git/GitHub, NuGet Package Manager.
